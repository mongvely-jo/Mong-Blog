CSS 링크 가상 클래스 (link, visited, hover, active)

2019 / June

개요

" :link , :visited , :hover , :active 링크는 마우스로 해당 객체위에 마우스를 올려놓았을때 변화를 주어 엑티브한 페이지를 구성할때 사용한다.
(링크뿐만 아니라 모든 요소에서 사용 가능)

관련태그

셀렉터 의미 예시 특징
link 아직 방문하지 않은 링크 a:link {color:green;} a 태그에만 적용 가능
visited 이미 방문한 링크 a:visited {color:green} a 태그에만 적용 가능
hover 마우스를 올렸을때 a:hover {color:red;}
active 마우스로 링크를 클릭하는 중일때 a:active {color:yellow;}
.mylink :hover { background-color: yellow; }
.mylink :active { background-color: red; }
.mylink :link { color: green; }
.mylink :visited { color: hotpink; }
